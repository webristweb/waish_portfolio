# Ankit Nayak - Portfolio Website

A modern, responsive portfolio website built with React and Vite, showcasing skills, projects, education, and contact information for a Full Stack Developer.

- Smooth scroll indicator
- Technology stack badges (Python, Django)

### 2. **About Section**

## 📦 Installation

1. **Clone the repository**
```bash
git clone <repository-url>
cd ankit-portfolio
```

2. **Install dependencies**
```bash
npm install
```

3. **Run development server**
```bash
npm run dev
```

4. **Build for production**
```bash
npm run build
```

5. **Preview production build**
```bash
npm run preview
```
