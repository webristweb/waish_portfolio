## 🚀 How to Run

### Option 1: XAMPP (Recommended)
```bash
1. Copy folder to: C:\xampp\htdocs\portfolio
2. Start Apache in XAMPP
3. Open: http://localhost/portfolio
```

### Option 2: PHP Built-in Server
```bash
php -S localhost:8000
# Open: http://localhost:8000
```

### Option 3: Live Server (VS Code)
```bash
1. Install Live Server extension
2. Right-click index.html
3. Select "Open with Live Server"
```

## 🎨 Customization Guide

### Change Personal Info
Edit `js/data.js`:
```javascript
// Update your details here
const skillCategories = [...];
const projects = [...];
const educationData = [...];
```

### Change Colors
Edit `css/style.css`:
```css
:root {
    --primary-color: #6366f1;  /* Your color */
    --accent: #22d3ee;         /* Your accent */
}
```

### Change Email
Edit `config.php`:
```php
define('CONTACT_EMAIL', 'your-email@example.com');
```

### Add Your Photo
Replace in `index.html`:
```html
<img src="your-photo.jpg" alt="Your Name">
```

## 🎯 Key Differences from React Version

| Feature | React Version | HTML Version |
|---------|--------------|--------------|
| Framework | React + Vite | Pure HTML/CSS/JS |
| Dependencies | npm packages | None |
| Build Process | Required | Not required |
| Server | Node.js | Apache/PHP |
| Contact Form | Would need backend | PHP included |
| Loading | Slower (bundle) | Faster (direct) |
| SEO | Needs SSR | Native support |
